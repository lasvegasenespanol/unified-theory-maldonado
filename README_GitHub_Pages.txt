Steps (no command line):
1) Create the repository here: https://github.com/new — name it: unified-theory-maldonado
2) Upload everything from this folder:
   - index.html
   - /assets (all files inside)
3) Turn on GitHub Pages:
   - Repo → Settings → Pages
   - "Deploy from a branch" → Branch: main  •  Folder: /(root)
   - Save and wait ~1–3 minutes.
4) Your site will appear at:
   https://lasvegasenespanol.github.io/unified-theory-maldonado/

Command‑line publish (copy/paste):
  git init
  git add .
  git commit -m "Initial publish: Unified Brane‑Cosmology"
  git branch -M main
  git remote add origin https://github.com/lasvegasenespanol/unified-theory-maldonado.git
  git push -u origin main
  # Then enable Pages: Settings → Pages → Deploy from branch (main / root)
