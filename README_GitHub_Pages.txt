
How to Publish This Site (GitHub Pages) — August 09, 2025

1) Create a new GitHub repository (e.g., unified-theory-maldonado).
2) Upload the contents of the 'unified_theory_site' folder (index.html + assets directory).
3) In the repo settings, enable GitHub Pages: Settings → Pages → Source: 'main' branch / root.
4) Wait ~1–3 minutes; your site will be live at https://<your-username>.github.io/<repo-name>/
